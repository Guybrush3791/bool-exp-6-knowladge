# MacOS
## Installare Brew (se non presente)
Dare il seguente comando da terminale
```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Installare Git
Dare il seguente comando da terminale
```
brew install git
```
